<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/AIDIT-A-Humanitarian-Cause/Aidit_Hackathon.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


<!-- Deployment -->
### :triangular_flag_on_post: Deployment
